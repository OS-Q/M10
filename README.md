[cppcheck](http://cppcheck.net/) 是静态的 C/C++ 代码分析工具，用以检查内存泄漏、错配的内存分配和释放、缓冲区溢出，以及更多问题。是静态的 C/C++ 代码分析工具，用以检查内存泄漏、错配的内存分配和释放、缓冲区溢出，以及更多问题。不同于C/C++编译器及其它分析工具，Cppcheck只检查编译器检查不出来的bug，不检查语法错误。
